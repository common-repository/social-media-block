Social media block for Gutenberg.

== Description ==
Drag & drop block in Gutenberg to place social media buttons on your site. 

The plugin allows you to place the following icons: 

* Facebook
* Twitter
* Google+
* Pinterest
* LinkedIn
* Tumblr
* VK
* Reddit
* OK
* StumbleUpon
* Telegram

You can pick from three different design styles: 
* Round
* Cubes
* Rectangle

You can also define the text above your icons, e.g. "Please share us!"

== Installation ==
* Install the plugin
* Activate the plugin
* Go to your Gutenberg-editor, click on the block-icon, and then select the "Social media share"-block
